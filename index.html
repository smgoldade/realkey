About realkey

realkey is a system for generating 3D models of keys that can be used for 3D printing keys.

realkey is powered at its core by build123d, a code-based CAD system written in Python.

Many of the target locks have very high tolerances, to improve print quality, some recommended settings are available:

3D Printing Settings Good plastics for keys are rigid and wear resistant. If it's good for gears, it's good for keys!

Plastics:

  • Carbon Fiber Nylon
  • Polycarbonate
  • PETG
  • PLA - Even PLA works well! It can be a little brittle however.

Settings:

  • Nozzle: 0.4mm
  • Start with your printers HIGHEST DETAIL default, then adjust
  • Layer Height: 0.08mm
  • First Layer Height: 0.16mm - 0.20mm
  • Top Surface Line Width: 0.32mm
  • Walls Printing Order: Outer/Inner
  • External Bridge Density: 107%
  • Wall Loops: 3
  • Top Shell Layers: 9
  • Top Surface Pattern: Monotonic Line
  • Infill: 100%
  • Infill Pattern: Gyroid - This probably does not matter!
  • Outer Wall Speed: 50mm/s
  • Inner Wall Speed: 80mm/s
  • Bridge External Speed: 10mm/s
  • Supports: Enabled
  • Ignore Small Overhangs: Disabled
  • Top Z Distance: 0.16mm-0.20mm
  • Bottom Z Distance: 0.20mm

For more information, checkout the github page!
